Proses dan Komputasi komputer

Komputer merupan benda 'bodoh', yang cepat, sangat cepat untuk memproses milyaran tahapan tanpa kita sadari. Komputer pada dasarnya hanya menjalankan intruksi yang kita tuliskan secara detil tahap demi tahap.

Sebagai contoh mari kita demontrasikan seberapa cepat komputer dapat bekerja menggunakan program simple 


function someFunction() {
        var iterasi = document.getElementById("iterasi").value
        var t0 = performance.now()
        var total = 0;
        for (var i = 0; i < iterasi; i++) {
            total++;
        }
        var t1 = performance.now()
        alert((t1 - t0) + " milisecond  denga hasil " + total)
    }

Silahkan mencoba dengan mengganti jumlah iterasi yang diinginkan, waktu untuk sekali proses tergantung dari kemampuan proses komputer yang digunakan.
